🗂️ Category Display Settings

By default, if a client can see more than one category, they're combined into one "Live Classes" tile — this makes sense for Monday + Wednesday classes. But standalone programs like Foundations or a Bodyweight Program shouldn't get lumped in. Check any category that should always show as its own separate tile, never combined.
